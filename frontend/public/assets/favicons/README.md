# Favicons

Keep each favicon design in its own folder:

```text
favicons/
  aisha-brain/
  future-icon/
```

To switch designs, update the favicon paths in `frontend/index.html`.

Each favicon folder can contain:

- `favicon.ico`
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png`
- `icon-192.png`
- `icon-512.png`
- `site.webmanifest`
